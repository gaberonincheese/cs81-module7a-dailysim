# Reflection – Daily Schedule Simulator

## What was your approach to designing the schedule?
Explain how you chose your activities and what made them personal or interesting.

I took one of my typical days during the summer session and translated some of my daily tasks to an array I created.  

## What was one challenge or unexpected behavior you encountered?
Describe anything surprising that happened when you tested the timing.

I really struggled to figure out how to use setTimeout, and this was because I was returning my outputtedText variable before running the function in setTimeout. I also had many minor problems with the timing again as I tried to figure out how to structure my code in such a way that each item in the array return on a new line.

## What does this assignment teach you about async code?
Compare this to a regular script that runs top to bottom without delays.

It enhances the flexibility of the code in a different way compared to the usage of functions. The capacity to be able to run multiple aspects of the program at once really benefits the complexity of things like web browsers, especially when there is no dependency between blocks of code. Obviously, introducing more complexity to code can cause problems, but when utilized effectively, the benefits far outweigh potential time-consuming costs.

## What creative element did you add?
Did you add randomization, surprise content, a twist, or other enhancements?

I added a function that allows for the inclusion of new, random tasks, then reorganizes the array given the new task in mind. Taking myself into consideration, it's a low-stakes way to introduce some variety into my day without having to make choices about what to do and without having to succumb to the same routine every day. 

## How does this project simulate or differ from real-world schedules?
Explain how well your simulation represents how time works in real life.

There's a clear chronological ordering to the schedule I made, which falls in line with how schedules work in the real-world. Still, the introduction of small pockets of variety account for the possibility to make new choices and keep things interesting in every day life.
